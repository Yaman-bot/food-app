# food-app

Install Dependencies (frontend & backend)\
<code>
npm install  \
cd frontend
npm install
</code>

# Run frontend (:3000) & backend (:5000)
<code>
npm run dev
</code>
# Run backend only
<code>
npm run server
</code>
Build & Deploy <br/>
# Create frontend prod build
<code>
cd frontend
npm run build
</code>
